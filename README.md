# Информационная подсистема ведения голосования

## Описание проекта
Java EE веб-приложение для проведения голосований. Проект разработан в рамках курсового проекта.

## Технологии
- Java EE 8 (Web Profile)
- JPA / Hibernate
- Servlets / JSP / JSTL
- PostgreSQL
- Maven
- JUnit / Mockito

## Структура базы данных
- Vote (голосования)
- Question (вопросы)
- User (пользователи)
- Choice (результаты голосования)

## Функциональность
- CRUD операции для всех сущностей
- Аутентификация пользователей
- Защита веб-запросов
- Статистика на главной странице

## Установка и запуск

### Требования
- JDK 11+
- Apache Maven 3.6+
- PostgreSQL 12+
- Сервер приложений (TomEE / Payara / WildFly)

### Настройка базы данных
1. Создайте базу данных `voting_db` в PostgreSQL
2. Настройте подключение в `src/main/resources/META-INF/persistence.xml`

### Сборка и деплой
```bash
mvn clean package
